# Требования к ML компонентам

## Модели
- **LLM**: deepseek-r1:8b (через Ollama)
- **Embeddings**: all-minilm-l6-v2 (для векторной БД)

## Производительность
- **Latency**: < 30 секунд на запрос
- **Throughput**: 10+ запросов в минуту
- **Время генерации теста**: < 2 минут
- **Время генерации плана урока**: < 1 минуты

## Метрики качества
- **Accuracy**: точность генерации структурированного JSON
- **BLEU/ROUGE**: для оценки качества текстового контента
- **JSON Validity Rate**: процент валидных JSON ответов
- **Content Relevance**: релевантность интересам ученика

## Ограничения
- **Память**: < 8GB RAM для модели
- **Стоимость**: бесплатно (Ollama локально)
- **Приватность**: все данные обрабатываются локально
